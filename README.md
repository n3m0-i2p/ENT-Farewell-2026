# ENT Farewell 2026

> **Electronics Farewell 2026** — Official premium farewell registration portal for BPI Electronics Technology department, Session 2022-23.

## Features

- 🎓 Multi-step registration wizard (Manifesto → Details → Customization → Checkout)
- 💳 Simulated bKash payment gateway with OTP & PIN verification
- 🎫 Digital receipt pass with downloadable PNG
- 💬 Opinion Wall for farewell messages
- 🔐 Admin Dashboard with password-protected edit/delete actions
- 📊 Real-time system activity logs
- 💾 100% client-side — runs entirely on `localStorage` (no database required)

## Tech Stack

- **HTML5** + **CSS3** + **Vanilla JavaScript**
- **Google Fonts** (Plus Jakarta Sans)
- **Zero dependencies** — single self-contained `index.html`

## Deployment

This project is Vercel-ready. Simply connect your GitHub repo to Vercel and it auto-deploys.
